ls - смотреть содержимое текущую директорию

cd -  меняет текущую директорию

man 
info
help справочники


. - ссылка на текущую папку (Я для папки)
.. - ссылка на родидельскую папку 

mkdir  - создает папки
touch создает пустой файл
rm - удаляет файлы и папки (папки только с параметром -r)

C - create | Создать   |  mkdir, touch, > , >> 
R - read   | прочитать | cat
U - update | обновить  | > , >> 
D - delete | удалить   | rm



cat - просмотр содержимого файла


ls > file.txt - записывает инфу из стандартного потока вывода в файл затирая уже существующую в файле инфу
x >> y - записывает инфу из стандартного потока вывода в файл не затирая существующие данные


cp <путь_к_файлу_который _копируем> <путь куда копировать> - копирует файл(ы) в указанное место
mv <путь_к_файлу_который_перемещаем> <путь куда переместить> - перемещает файл(ы) в новое место

ln - создает ссылку на файл ( твердая по умолчанию, -s для мягкой)

Шаблоны

* - любое количество симовлов (включая пустоту)
? - любой 1 символ
[] - один символ из перечисленный в скобках 
! - логическое отрицание
