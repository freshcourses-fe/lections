git init -  создает локальный репозиторий. Запускать только в той папке где находятся файлы проекта.

git add <имя_файла> - добавлфяет файл в зону индексации
git add . добавит все файлы которые найдет в индекс
git status показывает текущий статус

git commit сохраняет изменения в новый коммит
git commit -m 'Название комита' позволяет сразу же написать сообщение комита
git commit --amend - исправляет последний комит информацией из зоны индексации и его описание


git log показывает все комиты 
git log --all 
git diff показывает изменения в модифицированных файлах
git diff --staged показывает изменения в индексированных файлах

git reflog показывает все ваши локальные действия

git remote add origin <адрес> связывает локальный репозиторий с удаленным

git remote -v проверить какие репозиториии всязаны с удаленными

git push origin main - отправляем наши файлы в удаленный репозиторий origin на ветку main

При пуше в консоли оно попросит имя и пароль но вместо пароля ему надо указать специальный токен
сгенерировать его можно по следующему пути:
Settings -> Developer Settings -> Personal access tokens -> generate new token

токену можно указать имя и срок годности (no expiration - без срока годности)
В правах поставить галочку на repo и все.
Токен вам покажется один раз после создания поэтому его легче куда то сохранить

git clone <адрес> - скачать веcь репозиторий на новом компе
