                                Процессы

ctrl + c  - завершение процесса
ctrl + z - остановка процесса

jobs - показывает процессы для текущего терминала

fg <число> - выводи на передний план блоркирует терминал
bg <> -  выводи на задний план не блоркирует терминал

запуск в фоне - xeyes &

ps - команда для просмотра процессов
ps axu - для просмотра всех процессов в системе

kill -<signal> <id процесса> убивает указанный прцоесс
сигналы:
-9 безусловное завершение
-15 кнопочка выхода
killall <signal> <имя процесса> убивает все процессы у указанным именем

                                Установка ПО
                                
.deb - установочные фалйики для Дебиана и его детей

apt - установщик пакетов 

sudo apt install имя_программы - установка имя_программы, также используется для точечного обновления программы
sudo apt remove / purge имя_программы - удаляет программы с / без настроек и доп файлов

sudo add-apt-repository - добавить новый репозиторий

apt update - одновить список репозиториев и проверить обновления программ

apt list показывает список всех установленных пакетов
apt list --upgradable - показывает список пакетов которые можно обновить

apt upgrade - установить все обновления

apt autoremove - удяляет ненужные пакеты
