env - показывает все переменные окружения
В ней самые интересные вещи
    PATH - где ищутся команды и файлы
    SHELL - показывает текущую оболочку
    USER - ваше имя пользователя
    PWD , OLDPWD - текущая и передыдущая папка

Переменные можно посмотреть не лезя в env

echo - выводит текст в консоль

echo $SHELL - для выводя реузльтата переменых окружения в консоль

echo $(env) - для вывода результата команд в консоль

nano <путь к файлу> - текстовый редактор в термминале

<команада 1> | <команда 2> оператор перенаправления передает на вход команды 2 вывод команды 1

sort [пути к файлу] - соритрует содержимое файла или если файл не дали то сортирует ввод команды

grep <шаблон> - фильтрует данные которые ему поступают на вход по некоему шаблону и возвращает их в консоль

locate <шаблон> - ищет в файловой системе фйлик по шаблону (глоб) из кеша. Быстрый.
Для обновления кеша вручную можно использовать sudo updatedb.

find [-name <имя файла>] -  ищет в файловой системе обходя все папки

Права доступа

рут - суперпользователь, админ в линуксе который может делать все что угодно

sudo - исполнить команду от пользователя если не указать от какого то посчитает что от рута

su - сменить пользователя, если не указать имия то попробует сменить на рута

ls -l в первом столбце выводит вам 3 тройки прав доступа
rwx - чтение, запись, выполнения

1 тройка - права владальца файла
2 тройка - права группы владельца
3 тройка - права других пользователей

изменить права можно с помощью команды chmod:

chmod [ugoa][+-=][rwx] <файл которому меняем права>
